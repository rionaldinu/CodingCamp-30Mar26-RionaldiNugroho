# Expense & Budget Visualizer

## Description
A mobile-friendly web application to track daily expenses and visualize spending by category.

## Features
- Add and delete transactions
- Real-time total balance update
- Spending visualization using pie chart
- Data persistence using LocalStorage
- Sort transactions by newest or amount
- Dark/Light mode toggle

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- Chart.js

## How to Use
1. Enter item name and amount
2. Select category
3. Click "Add Transaction"
4. View total balance and chart
